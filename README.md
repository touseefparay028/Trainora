# Trainora
Trainora is a cutting-edge learning platform made by me for the college MCA project for fulfilling the requirement for completion of the degree

Trainora LMS is a modern **Learning Management System** designed for schools, colleges, and training institutions.  
It offers **real-time progress tracking**, role-based access, course management, and smooth communication between students and teachers.

---

## 🚀 Features

- **Role-based Access Control**  
  - Admin, Teacher, and Student roles.
  
- **Course Management**  
  - Create, edit, delete, and assign courses.
  
- **Assignment & Submission Tracking**  
  - Students can submit work, teachers can grade instantly.

- **📈 Real-time Progress Tracking**  
  - Live updates on student learning progress using modern web technologies.

- **Responsive Design**  
  - Built with **Bootstrap 5** for mobile-first experience.

- **Animated UI**  
  - Smooth animations using AOS for a modern feel.
